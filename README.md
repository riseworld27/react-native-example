## Getting Started

Please follow the below instruction get it running.

### Prerequisites

```
node > 8.x
yarn > 1.x
```

### Installing

```
npm install -g yarn
yarn
```

### iOS

Please open ZirtuePayee.xcodeproj in src/ios
Run the project in xcode
